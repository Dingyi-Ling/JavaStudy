
# JavaStudy

Welcome to the JavaStudy repository. This project is primarily focused on Java and includes various course materials and project developments.

## Repository Structure

- **.gitignore**: Defines files and directories to be ignored by Git.
- **README.md**: This file, providing an overview of the project.
- **SMS/**: Contains subdirectories for different academic years:
  - **2024_2025_JavaCourse/**: Course materials for the Java course in the academic year 2024-2025.
  - **2025_2026_ProjectDevelopment/**: Project development materials for the academic year 2025-2026.

## Languages

This repository primarily uses the following languages:

- **Java**: 79.7%
- **Jupyter Notebook**: 20.3%

## Installation

To get started with this project, clone the repository:

```bash
git clone https://github.com/Dingyi-Ling/JavaStudy.git
```

## Usage

Navigate to the appropriate directory for the course or project you are interested in. Open the Java files or Jupyter Notebooks to view the content.
